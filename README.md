# 🌧️ Rainfall Prediction App

A Machine Learning based web application developed to analyze rainfall patterns and predict rainfall-related outcomes using historical flood and rainfall datasets.

This project was created as part of the **SmartBridge Internship Program** to gain practical experience in Python, Machine Learning, and real-world data analysis.

---

## 🚀 Features

- 📊 Rainfall prediction using ML models
- 📁 Dataset analysis and preprocessing
- 🧠 Machine Learning model training
- 🌐 Simple Python-based web application
- 📈 Flood and rainfall data handling
- ⚡ User-friendly interface

---

## 🛠️ Technologies Used

- Python
- Machine Learning
- Pandas
- NumPy
- Scikit-learn
- Flask
- Excel Dataset Handling


## ⚙️ Installation & Setup

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/Akhila-Palagani24/Rainfall-Prediction-App.git
```

### 2️⃣ Navigate to Project Folder

```bash
cd Rainfall-Prediction-App
```

### 3️⃣ Install Dependencies

```bash
pip install -r requirements.txt
```

### 4️⃣ Run the Application

```bash
python app.py
```
