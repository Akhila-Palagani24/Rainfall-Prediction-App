# 🌊 Rising Waters: Flood Prediction using Machine Learning

## 📌 Overview
This project predicts flood occurrence using environmental data like rainfall, river level, temperature, and humidity.

## ⚙️ Tech Stack
- Python
- Scikit-learn
- Flask

## 🚀 How to Run

1. Install libraries:
pip install -r requirements.txt

2. Train model:
python train_model.py

3. Run app:
python app.py

## 📊 Sample Input
{
  "rainfall": 150,
  "river_level": 7,
  "temperature": 30,
  "humidity": 85
}

## 📤 Output
Flood Likely / No Flood
